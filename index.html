About Me

I am a master's student in Medical Informatics with a background in nursing. My work focuses on applying artificial intelligence and clinical data analysis to healthcare problems, including pulmonary function testing, medical NLP, and retrieval-augmented generation systems for patient education.

Research Interests

My research interests include medical artificial intelligence, clinical natural language processing, pulmonary function interpretation, spirometry flow-volume loop analysis, and AI-assisted patient education systems.

Selected Projects

PRISm and Inspiratory Flow-Volume Loop Analysis

This project investigates preserved ratio impaired spirometry using pulmonary function test data, with a focus on inspiratory flow-volume loop dynamics and upper airway dysfunction-related features.

Deep Learning Classification of Spirometry Flow-Volume Loop Morphology

This project applies a ResNet-18 convolutional neural network to classify spirometry flow-volume loop morphology and explore AI-assisted interpretation of pulmonary function testing.

TMUH CareGuide AI: RAG-Based Patient Education Chatbot

This project implements a retrieval-augmented generation chatbot for patient education using hospital-provided health education documents as the knowledge base.

Medical Text Classification

This project explores clinical text classification using TF-IDF, transformer-based models, and LLM-assisted labeling strategies for medical condition classification.
