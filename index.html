ML-Based Inspiratory Flow–Volume Loop Analysis: UAO Detection & PRISm Phenotyping

First Author · APSR

Study 1 — CNN

ResNet-18 on 154 spirometry loop images. Binary UAO vs non-UAO classification with Grad-CAM error analysis.

AUC 0.807 Hold-out AUC · Macro F1 0.742

Study 2 — PRISm Phenotyping

7,187-patient cohort. Logistic regression + XGBoost + GMM clustering on inspiratory spirometric indices.

OR 1.56 Stable PRISm → UAW proxy positivity (adj.)

Key Finding

Expiratory features alone cannot reconstruct inspiratory-limited physiology (AUC 0.557–0.568). GMM identified a distinct inspiratory-limited cluster (30.3%, n=443) enriched for Stable PRISm.

Dataset

TMUH spirometry database · 57,295 records · Dec 2007 – May 2026 · ResNet-18 / ResNet-50 / EfficientNet-B0 / DenseNet-121 compared

Biomedical Text Classification — Kaggle Competition

NLP Competition

Problem

Classify 14,000+ medical condition descriptions into five disease categories from noisy, imbalanced labels.

Approach

Benchmarked TF-IDF baselines (NB, LR, SVM). Fine-tuned PubMedBERT and BioLinkBERT-large with 5-fold CV. Applied ensemble learning, overlap-aware post-processing, and uncertainty-based relabeling.

Key Result

0.71760 Public leaderboard score · 5-model BioLinkBERT-large ensemble

Stack

PubMedBERT · BioLinkBERT · TF-IDF · Ensemble Learning · HuggingFace

AI-Driven Pre-Hospital Handover Automation

Data Mining Course

Problem

Clinical handovers from EMS to ED are prone to information loss and communication gaps, contributing to medical error.

Approach

Multimodal pipeline combining Speech-to-Text and NLP. Extracted structured clinical entities from unstructured handover narratives.

Result

Prototype validated on a test dataset for pipeline feasibility. Demonstrates end-to-end structuring of verbal handover data.

Impact

Reduces medical errors by improving EMS-to-ED data continuity in high-pressure handover environments.
