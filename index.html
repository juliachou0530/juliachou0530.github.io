Selected Projects

PRISm and Inspiratory Flow-Volume Loop Analysis

This project investigates preserved ratio impaired spirometry using pulmonary function test data, with a focus on inspiratory flow-volume loop dynamics and upper airway dysfunction-related features.

Deep Learning Classification of Spirometry Flow-Volume Loop Morphology

This project applies a ResNet-18 convolutional neural network to classify spirometry flow-volume loop morphology and explore AI-assisted interpretation of pulmonary function testing.

TMUH CareGuide AI: RAG-Based Patient Education Chatbot

This project implements a retrieval-augmented generation chatbot for patient education using hospital-provided health education documents as the knowledge base.

Biomedical Text Classification — Kaggle Competition

Developed a biomedical NLP pipeline to classify medical condition descriptions into five disease categories using traditional machine learning, biomedical transformer models, and LLM-assisted labeling strategies.

  • Processed 12,994 training samples and 1,444 testing samples, including label encoding, class distribution analysis, token-length analysis, and train-test overlap investigation.
  • Benchmarked TF-IDF-based models including Naïve Bayes, Logistic Regression, SVM, Decision Tree, and KNN.
  • Fine-tuned PubMedBERT and BioLinkBERT-large for biomedical text classification using 5-fold cross-validation.
  • Applied ensemble learning, overlap-aware post-processing, and uncertainty-based relabeling to improve performance on noisy medical labels.
  • Achieved a best public leaderboard score of 0.71760 using a five-model BioLinkBERT-large ensemble.
